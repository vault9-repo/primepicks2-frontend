# primepicks2-frontend

Local dev:
1. npm install
2. npm run dev  -> open http://localhost:5173 (Vite serves `public/index.html`)

Build:
1. npm run build
2. Dist output in `/dist` (deploy this to Render / Netlify / etc.)

Put your images in `public/assets/`:
- BACKGROUND.jpg
- logo.png
- top1.PNG .. top4.PNG
- logos/TEAMNAME.png
